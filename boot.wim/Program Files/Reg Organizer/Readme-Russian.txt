  Описание программы
  ------------------

  Многофункциональная программа для редактирования, чистки и
  обслуживания реестра, исправления ошибок в системе и повышения
  ее производительности. Функция глубокого поиска по реестру
  позволит отыскать все ключи реестра, относящиеся к интересующему
  приложению. Программа позволяет редактировать файлы реестра
  (.reg) и просматривать их содержимое прямо из Проводника Windows.
  Содержит деинсталлятор приложений, позволяющий полностью
  удалять ненужные программы из системы. Ряд других функций будет
  полезен как администраторам, так и обычным пользователям.
