Каталог для основной (MAIN) и дополнительных баз называется SHA (в каталоге uVS).
Дополнительные базы загружаются автоматически и только в R/O режиме.

Загрузка происходит as is т.е. все _дополнительные_ базы загружаются в отдельный массив данных
находящийся в оперативной памяти БЕЗ анализа их содержимого на пересечение между собой и 
пользовательской базой.

Добавить хэш в пользовательскую базу возможно лишь при отсутствии хэша во ВСЕХ загруженных
базах.
Удалить хэш можно лишь из пользовательской базы проверенных файлов.

Т.е. схема использования набора баз простая:
Каждый кто хочет выложить свою базу для общего пользования называет ее уникальным образом,
пользователь собирает дополнительные базы в каталог SHA, соотв. пользователь имеет собственную
доступную для изменений базу (см. Sha1Name в settings.ini) и набор R/O баз за редактирование
которых отвечают уже их создатели.
