Media content 0.6.1 - плагин содержимого для Total Commander 6.5

Для корректной работы плагина необходим DirectX 8.1 или выше.

История

Плагин определяет информацию о медиа-файлах:
  Продолжительность в формате времени
  Продолжительность в формате ms (милисекунды)
  Аудио кодек потока
  Видео кодек потока
  и т.д.
