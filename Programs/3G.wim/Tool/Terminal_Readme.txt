Huawei Terminal — программа терминала для прямой работы с беспроводными модемами Huawei, а также (ограниченно) с любыми другими модемами.

подключаться нужно не на порт модема, а на порт 3G PC UI Interface! (обычно имеет номер, на 1 больше, чем порт модема.)

команды управления диапазонами для модемов huawei:
AT^SYSCFG=13,1,3FFFFFFF,2,4 (только GPRS/EDGE)
AT^SYSCFG=14,2,3FFFFFFF,2,4 (Только 3G/WCDMA)
AT^SYSCFG=2,1,3FFFFFFF,2,4 (предпочитать GPRS/EDGE)
AT^SYSCFG=2,2,3FFFFFFF,2,4 (предпочитать 3G/WCDMA)

Все права на программу принадлежат Александру С. Шокину.

Программа бесплатна для любого *некоммерческого* использования. По вопросам коммерческого использования обращайтесь к Автору.

Текущая версия программы указана в файле history.txt

Программа написана на VisualBasic.NET (2008) и требует в net.framework 2.0 для работы.
